Navigating a Jupyter Notebook

Jupyter Notebook is built off of IPython, an interactive way of running Python code in the terminal using the REPL model (Read-Eval-Print-Loop). A Jupyter Notebook goes by the extension ".ipynb".
Please check the previous sections on how to launch a Jupyter Notebook. Once launched, you can navigate through the file system and open an existing notebook, or create a new notebook by navigating to the desired loaction in the file system and clicking on the "New" button on the right-hand side and then "Python 3".
  • On the top you can see when the last checkpoint was made/notebook was saved.
  • Below that, we have the Menu Bar with the basic menus with general options.
  • Below that, we have various buttons which correspond to save, insert cell, cut cell, copy selected cells, paste cells, move cells up or down, run, interrupt, restart, restart and re-run the kernel. Beside that, we have various cell options like Code and Markdown.
  • Next, we have the main area where we write our code. Usually, each line of code is written in a single cell, except for classes and functions which can have many lines of code but still use a single cell.
  • You can write code in these cells and press Shift + Enter/Return to run them.

Running Python on Command Line

This requires the user to open the Anaconda Prompt/Terminal.
  • Executing .py files on command line
    1. Navigate to the folder where you have saved the python file.
    2. Initially you will be in the root directory. To jump into a sub-directory, you can type in "cd directory-name" and press Enter/Return.
    3. You can jump out of a directory to the parent directory by typing "cd .." and pressing Enter/Return.
    4. Once you are in the directory that contains your python file with the ".py" extension, you can type "python3 filename.py" to run it.

  • Using the python interpreter
    1. The Interpreter does not require you to be at a specific location in the file system.
    2. You can simply type in "python3" in the Anaconda Prompt or Terminal to open the Interpreter.
    3. Type in the commands you want to execute and press Enter/Return after typing each one of them.
    Extra: For lines of code requiring indentations, e.g., function bodies, loop bodies, you can press the Tab key to give indentations for each line as needed.

Some additional tips

  • Organize your code in a separate workspace, into proper folders for ease of access and better navigation.
  • Most of the coding in this course will be done on Jupyter Notebooks but it is always recommended to explore other forms that suit you.
  • There are certain IDEs like PyCharm and text editors like VSCode, Sublime Text and Atom that are well suited for coding and are really helpful with their own set of features.
  • Google Colab is also an interesting resource that allows you to run code on their GPUs and TPUs. You can also work on datasets stored on your Google Drive!

Using Anaconda virtual environment on Windows/Mac/Linux (Optional)

For this part, the Command Line version of Anaconda needs to be installed on your system. Please go through the previous section for doing so.
  1. Check if conda is installed and is in your PATH. Open your Anaconda Prompt/Terminal and type in "conda -V" and press Enter/Return. If conda is installed, you should see the conda version in response.
  2. Check if conda is up to date. Type "conda update conda" and press Enter/Return. This will update conda if any updates are available.
  3. Type "conda create --name yourenvname" and press Enter/Return. Press "y" to proceed.
  4. Activate your environment by typing in "conda activate yourenvname" and pressing Enter/Return.
  5. To install additional packages only to your virtual environment, use the format "conda install -n yourenvname [package]"
  6. Deactivate your virtual environment after use by typing "conda deactivate" and pressing Enter/Return.
  7. You can delete your environment by typing "conda remove -n yourenvname -all" and pressing Enter/Return.
